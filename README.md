# 🚀 Pro-Manage | Plateforme de Gestion de Projets Freelance

**Pro-Manage** est une application web Full-stack conçue pour aider les freelances et les étudiants à organiser leurs projets, suivre leurs tâches et gérer leur budget de manière centralisée.

---

## 📋 Fonctionnalités
- **Tableau de Bord Interactif :** Visualisation en temps réel du nombre de projets actifs, du budget total et des tâches restantes.
- **Gestion CRUD Complète :** Création, modification et suppression de projets et de tâches.
- **Suivi Budgétaire :** Calcul automatique du budget cumulé des projets.
- **Organisation par Statut :** Classification des projets (À faire, En cours, Terminé).
- **Architecture Relationnelle :** Base de données SQL structurée (Lien User-Projet-Tâche).

## 🛠️ Stack Technique
- **Frontend :** HTML5, CSS3 (Flexbox & Grid), JavaScript (Manipulation du DOM).
- **Backend :** PHP 8 (Utilisation de **PDO** pour la sécurité contre les injections SQL).
- **Base de données :** MySQL.
- **Icônes :** Font Awesome.


## 🚀 Installation Locale
1. Clonez le projet dans votre dossier `htdocs`.
2. Importez le fichier `database.sql` dans votre **phpMyAdmin**.
3. Configurez la connexion dans `includes/db.php`.
4. Accédez au projet via `localhost/pro-manage`.
